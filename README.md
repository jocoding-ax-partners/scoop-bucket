# scoop-bucket

> [Scoop](https://scoop.sh) manifests for [axhub](https://cli.jocodingax.ai) on Windows.

## Install

```powershell
scoop bucket add jocoding-ax-partners https://github.com/jocoding-ax-partners/scoop-bucket
scoop install axhub
```

## Manifests

- `axhub` — CLI for the axhub developer platform.

## Status

**v0.0.1** — manifest points at the live CLI CDN artifacts and validates against
the published release checksums. Future tags remain autoupdate-driven.
