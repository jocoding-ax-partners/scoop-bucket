# scoop-bucket

> [Scoop](https://scoop.sh) manifests for [axhub](https://cli.jocodingax.ai) on Windows.

## Install

```powershell
scoop bucket add jocoding-ax-partners https://github.com/jocoding-ax-partners/scoop-bucket
scoop install axhub
```

## Manifests

- `axhub` — CLI for the axhub developer platform.

## Status

The checked-in manifest currently targets **0.1.0** assets from
`https://cli.jocodingax.ai/0.1.0/`, while `checkver` and `autoupdate` continue
to follow `https://cli.jocodingax.ai/version.txt` and the published release
checksums for future tags.
